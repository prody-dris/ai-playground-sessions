Why CLI?

In Session 3, Claude UI could read your repos but couldn't write back — no commits, no PRs, no file updates. CLI changes that.

Claude UI

1

Reads repos via token

2

Great for thinking and writing

3

Cannot push or raise PRs

Claude Code (CLI)

1

Reads AND writes your files

2

Commits, pushes, raises PRs

3

Follows your rules automatically

Hooks, MCPs & More

Automated guardrails that run before and after every action. Tools that connect directly to your Slack, Gmail, Calendar. A guest who maintains production software with AI daily.

We'll set these up together on Thursday

Prerequisite: Set Up CLI On Popular Demand

Multiple people asked to get started early. One prompt sets up everything. Do this before Thursday and you'll hit the ground running.

1

Open claude.ai in your browser

2

Go to the Prompts page and copy the CLI Setup Guide

3

Paste it into Claude and follow along — 20-25 minutes

4

Bookmark the Claude Code Cheatsheet for daily reference

The prompt handles everything: GitHub verification, CLI install (through your Claude subscription, not npm), best practices setup, and your first demo run. Zero coding.
