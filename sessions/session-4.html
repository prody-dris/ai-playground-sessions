Session 3 proved you can build. Now we upgrade how you build. Claude Code (CLI) is the full-power version of Claude — it can read AND write to your repos, run commands, raise PRs, and automate workflows. Today, we set it up and learn to use hooks to make it even smarter.

The Big Idea

"The best tool is the one that disappears into your workflow." — Jef Raskin, creator of the Macintosh project

Claude UI is a conversation. Claude Code is a collaborator. It sits in your terminal, reads your files, writes code, runs tests, and pushes changes — all while following your CLAUDE.md and Security.md rules. The difference? UI is like texting a colleague. CLI is like pair programming with them.

Think of it this way

Claude UI is like calling someone on the phone to ask for help. Claude Code is like having them sit next to you, looking at the same screen, making changes alongside you.

What Are Hooks?

Hooks are automatic actions that run before or after Claude does something. Think of them as guardrails that activate themselves.

Think of it this way

You know how your car beeps when you don't wear a seatbelt? That's a hook. It doesn't stop you from driving, but it makes sure you're safe before you go. Pre-hooks check before action. Post-hooks verify after.

1

Pre-commit hook: Scans for secrets (API keys, passwords) before any code is committed. If it finds one — blocked.

2

Pre-push hook: Checks if Security.md rules are followed before pushing to GitHub.

3

Post-commit hook: Automatically updates your heartbeat.md with what just changed.

4

Custom hook: Runs your own validation — "does this PR have a description?", "did the tests pass?"

Hooks are why CLI is mandatory for serious work. They enforce discipline automatically. No human memory required.

What Are MCPs?

MCP stands for Model Context Protocol. In simple terms: it's how you connect your personal tools — Slack, Gmail, Calendar, Linear — directly to AI. Instead of copy-pasting between apps, AI reads from and writes to your tools natively.

1

Slack MCP: AI reads your Slack messages and drafts replies

2

Gmail MCP: AI can search your emails and draft responses

3

Calendar MCP: AI checks your schedule and suggests meeting times

4

Linear MCP: AI reads your tickets and creates new ones

We won't set up all MCPs today — just understand the concept. The goal is to know what's possible so you can decide what's useful for your workflow.
